Projects

Fruits for Harvest: First Solo Game

This was a Unity 2D platformer and the first game I made solo. While it's simple in concept, I learned a lot of C# and Unity coding principles from it.

Skills

Fungle Bungle: Home Alone

This was a group developed Unity 2D platformer that used AR recognition to load the game on a plane surface and read a physical card to alter the game state. I implemented the AR plane recognition and edited the Android XML files to enable cross-device compatability.

Skills
  • AR Foundations
  • Android XML Editing
  • Unity

Schrodunnit

This was a group developed Unity project that aimed to create a comic exploring the idea of Schrodinger's Cat. I wrote the script and implemented the dialogue system using the Ink plugin.

Skills
  • C#
  • Ink
  • Unity

Mobile Data Narrative Site

This was an HTML site I designed to explore the playing habits of Animal Crossing players. I used HTML, CSS, JavaScript, D3.js, and React.js to create a responsive site that displays the data in an interactive way.

Skills
